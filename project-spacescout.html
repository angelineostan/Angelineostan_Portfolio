SIA 101 Project

SPACESCOUT

Overview

SpaceScout is a high-fidelity UI/UX prototype designed to simplify the dorm-hunting experience for students. The project focuses on creating a "trust-first" interface where users can explore verified listings through a modern, clean, and intuitive mobile-responsive web environment.

The Design Challenge

The challenge was to organize heavy amounts of data—such as amenities, location maps, and pricing tiers—without overwhelming the user. I focused on visual hierarchy and a modular layout to ensure the property search felt effortless rather than exhausting.

Design Features

  • ▹ User-Centric Navigation: Tailored dashboards for both property owners and student seekers.
  • ▹ Consistent UI Kit: A comprehensive design system of reusable buttons, inputs, and cards.
  • ▹ Interactive Prototyping: Fully mapped user flows for listing properties and sending inquiries.
