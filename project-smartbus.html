PF102 Project

SMARTBUS

Overview

Literaria was developed to address the declining reading comprehension rates among students. As a Capstone project, the goal was to create an adaptive platform that gamifies the reading experience while providing teachers with real-time data on student progress.

The Challenge

The main technical challenge was creating an "Adaptive" system—meaning the difficulty of the text and quizzes needed to change based on the student's previous performance in real-time.

Key Features

  • ▹ Real-time comprehension tracking via Firebase.
  • ▹ Dynamic UI that adapts to different reading levels.
  • ▹ Teacher dashboard for class-wide analytics.

Technologies Used

  • HTML
  • Tailwind CSS
  • JavaScript
  • Flask (Python)
  • Firebase Firestore
  • Text-to-Speech (Gemini TTS)
  • Speech-to-Text (Google Cloud STT Chirp Model)
