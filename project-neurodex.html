SIA 102 Project

NEURODEX

Overview

NeuroDex is a specialized web platform designed to assist medical professionals in the detection and classification of brain tumors. By leveraging Deep Learning models, it provides a fast and reliable second opinion on MRI scans, streamlining the diagnostic workflow.

The Challenge

The primary challenge was integrating a complex Convolutional Neural Network (CNN) into a responsive web environment. Ensuring that MRI data could be processed by TensorFlow and results stored accurately in MySQL while maintaining a smooth UI was critical.

Key Features

  • ▹ Automated Brain Tumor Classification using TensorFlow/Keras.
  • ▹ Secure data persistence for patient records using MySQL.
  • ▹ Clean, responsive medical interface built with Bootstrap.
