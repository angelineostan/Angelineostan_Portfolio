IM101/CC106 Project

FUREVER HOME

Overview

Furever Home is a mobile application developed to simplify the pet adoption process. Built specifically for the Android platform, it connects animal shelters with potential adopters, allowing users to browse pets, submit applications, and receive real-time updates on their adoption status.

The Challenge

The primary technical challenge was implementing real-time data synchronization across multiple devices. Using Firebase Realtime Database, I had to ensure that pet availability and application statuses were updated instantly to prevent double-applications for the same pet.

Key Features

  • ▹ Real-time pet listings and availability sync via Firebase.
  • ▹ User authentication and personalized adoption profiles.
  • ▹ Admin portal for pet management and application tracking.
